# Trabajo Práctico (TP)

Este repositorio contiene el Trabajo Práctico correspondiente a la materia Desarrollo de Sistemas Web (Front End).  
El objetivo del TP es aplicar los conocimientos adquiridos en clase mediante un proyecto que integra teoría y práctica,
a la vez que el docente obtiene un diagnóstico de los conocimientos previos del grupo de estudio.


# Contenido del repositorio

- Código fuente 
- Carpeta de estilos CSS
- Carpeta de recursos gráficos (imágenes, íconos)
